Простое приложение для учета личных финансов, сделанное на .NET MAUI в рамках задания КТ-1.

## Описание

Приложение представляет собой основной экран для учета финансов. Реализован интерфейс с разделением на три части:

* верхняя часть — отображает название и общий баланс
* центральная часть — список последних операций
* нижняя часть — кнопки для действий

## Функционал

На данный момент реализовано:

* отображение общего баланса
* визуальный индикатор (ProgressBar)
* список транзакций (через CollectionView)
* кнопки:

  * Доход
  * Расход
  * Отчеты

Все данные пока тестовые и заданы вручную.

## Структура интерфейса

Экран разбит с помощью Grid на 3 области:

* 20% — заголовок и баланс
* 60% — список операций
* 20% — кнопки

## Используемые технологии

* .NET 10
* .NET MAUI
* XAML

## Как запустить

1. Открыть проект в Visual Studio
2. Убедиться, что выбран .NET 10
3. Нажать "Запуск"
4. Выбрать платформу (Windows или Android)
