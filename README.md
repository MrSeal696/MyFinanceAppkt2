Приложение для учета личных финансов, созданное на .NET MAUI. Проект выполнен в рамках заданий КТ-1 и КТ-2.

## Описание

Приложение позволяет вести учет личных финансов и работать с транзакциями. Реализован интерфейс с разделением на несколько страниц и навигацией между ними.

### Основные возможности

* Просмотр общего баланса и прогресса расходования средств
* Просмотр списка последних транзакций
* Добавление новых транзакций
* Просмотр отчетов
* Управление категориями транзакций
* Навигация между страницами через меню

## Структура интерфейса

### КТ-1 (интерфейс)

* Экран разделен на три области:

  * Верхняя: заголовок и баланс
  * Центральная: список последних операций (CollectionView)
  * Нижняя: кнопки действий (Доход, Расход, Отчеты)
* Используются элементы:

  * Label, ProgressBar, CollectionView, Button, Border (Frame устарел)

### КТ-2 (навигация)

* Добавлены страницы:

  * MainPage — главная
  * AddTransactionPage — добавление транзакции
  * ReportsPage — отчеты
  * CategoriesPage — управление категориями
* Настроен **AppShell** для навигации:

  * Flyout меню с переходами на все страницы
  * Автоматическая кнопка "Назад"
* Переходы по кнопкам реализованы через `Shell.Current.GoToAsync()`
* Зарегистрированы маршруты для страниц в AppShell

## Используемые технологии

* .NET 10
* .NET MAUI
* XAML

## Как запустить

1. Открыть проект в Visual Studio
2. Убедиться, что выбран .NET 10
3. Нажать "Запуск"
4. Выбрать платформу (Windows или Android)
